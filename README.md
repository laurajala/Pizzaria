# 🍕 Pizzaria - Sistema de Pedidos

Sistema web desenvolvido em PHP para simulação de pedidos de uma pizzaria, com painel administrativo para gerenciamento dos pedidos.

---

## 🚀 Sobre o projeto

Este projeto simula o fluxo completo de um sistema de pedidos de pizzaria, permitindo:

- Montagem de pizzas pelo cliente
- Seleção de borda, massa e até 3 sabores
- Registro de pedidos em banco de dados
- Painel administrativo para gerenciamento
- Atualização de status do pedido
- Exclusão de pedidos

---

## 🧰 Tecnologias utilizadas

- PHP
- MySQL
- PDO (PHP Data Objects)
- HTML5
- CSS3
- Bootstrap
- JavaScript (básico)

---

## ⚙️ Funcionalidades

### Cliente
- Montar pizza personalizada
- Selecionar borda e massa
- Escolher até 3 sabores
- Realizar pedido

### Administração
- Listar pedidos realizados
- Atualizar status do pedido
- Remover pedidos
- Visualizar detalhes da pizza

---

## 🗄️ Estrutura do sistema

- `index.php` → Página de criação de pedidos
- `dashboard.php` → Painel administrativo
- `process/pizza.php` → Lógica de criação de pedidos
- `process/orders.php` → CRUD dos pedidos
- `config/conn.php` → Conexão com banco de dados
- `templates/` → Estrutura de layout (header/footer)
